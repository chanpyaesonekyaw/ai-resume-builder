### Installation

1. Backend:
```bash
cd ai-resume-builder/backend
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Create a `.env.local` file in the root directory and add your environment variables:
```env
MONGO_URI=your_mongodb_url
PORT=your_port_number
```

4. Start the server:
```bash
npm start
```
